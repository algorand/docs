# Algorand Documentation Website

### Setup

- `pip install -r requirements.txt`

### Contributing
For more information about how to contribute documentation see the mkdocs documentation:
http://www.mkdocs.org/user-guide/writing-your-docs/

### Testing

```
mkdocs serve
```

### Deploying

```
mkdocs gh-deploy
```
